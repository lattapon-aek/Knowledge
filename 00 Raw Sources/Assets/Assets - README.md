# Assets

ใช้เก็บ assets ที่อ้างอิงร่วมกับ source ต้นฉบับ เช่น:
- รูปภาพจากบทความ
- screenshots
- diagrams
- attachments จาก web clipper

หลักการ:
- เป็นไฟล์ประกอบของ source ไม่ใช่ note ความรู้
- ถ้า asset ผูกกับ source ใดชัดเจน ควรตั้งชื่อให้ตาม source นั้น
