## หมายเหตุการใช้งาน

- bundle แบบนี้สะท้อนว่า note ปลายทางสังเคราะห์จากหลาย official docs
- ถ้าภายหลังต้องการ trace ให้ละเอียดขึ้น ควรแตกเป็น raw files จริงใน `Articles/` หรือ `Papers/`
- สำหรับ note ที่ใช้ official docs แบบ strict ควรคง `Official References` ภายในไฟล์ปลายทางต่อไป
