# การติดตั้งและเริ่มใช้งาน

---

## ติดตั้ง

```bash
# macOS / Linux / WSL (แนะนำ — auto-update)
curl -fsSL https://claude.ai/install.sh | bash

# Windows PowerShell (auto-update)
irm https://claude.ai/install.ps1 | iex

# Homebrew (ไม่ auto-update)
brew install --cask claude-code
```

> ℹ️ Native install จะ **อัพเดตอัตโนมัติ** ในพื้นหลัง

---

## ตั้งค่า API Key

```bash
# ใส่ไว้ใน ~/.zshrc หรือ ~/.bashrc เพื่อไม่ต้องตั้งค่าทุกครั้ง
export ANTHROPIC_API_KEY='sk-ant-...'
```

---

## คำสั่งที่ใช้บ่อย

```bash
claude                        # interactive mode
claude -p "review ไฟล์นี้"   # non-interactive (print mode)
claude -r                     # ต่อ session เดิม
claude -c                     # ต่อจาก message ล่าสุด
claude --version               # ตรวจสอบ version (ต้องใช้ v2.1.32+ สำหรับ Agent Teams)
```

---

## Requirements

- Agent Teams ต้องการ Claude Code **v2.1.32+**
- Agent Tool (เดิมชื่อ Task Tool) เปลี่ยนชื่อตั้งแต่ **v2.1.63**
