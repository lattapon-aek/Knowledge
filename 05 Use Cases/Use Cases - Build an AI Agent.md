## Design Checklist

- เป้าหมายของ agent คืออะไร
- อะไรควรเป็น tool และอะไรไม่ควร
- อะไรควรเป็น workflow ตายตัวแทน agent
- จะจำกัด cost, latency, และ risk อย่างไร
